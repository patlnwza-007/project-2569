## หลักการทำสไลด์ (อ่านก่อนเริ่ม)
- **1 สไลด์ = 1 ความคิดหลัก** — ถ้าเนื้อหายาวเกิน ให้แตกเป็นหลายหน้าแทนที่จะยัดใส่หน้าเดียว
- **ข้อความบนสไลด์ต้องสั้นกว่าที่พูด** — bullet ไม่เกิน **5 ข้อต่อสไลด์** ข้อละไม่เกิน 1 บรรทัด/8–10 คำ รายละเอียดยาวให้ใส่ใน **Script พูด** (= ช่อง Notes ของ Canva) แทน (เพดานเนื้อหาแบบละเอียดต่อชนิด ดู [`slide_content_generator.md`](../guide/slide_content_generator.md) §3)
- **ภาพ/แผนภาพแทนคำบรรยาย** เมื่อทำได้ (เช่น สไลด์สถาปัตยกรรม/ตารางเปรียบเทียบ ควรมี diagram/ตาราง ไม่ใช่ bullet list ล้วน)
- **คงคำศัพท์ให้ตรงกับ proposal_ch1.md** — ชื่อระบบ ชื่อ role ต้องเหมือนกันทุกที่ ไม่สลับคำ
- **ผู้ฟังต้องรู้ว่าอยู่ตรงไหนเสมอ** — มีหน้า Outline หลังหน้าปก, ทุกสไลด์เนื้อหามีเลขหัวข้อ (1–9) ตรงกับ Outline + Title/Subtitle ชัดเจน + footer เลขหน้า (วิธีทำดู `slide_generate_guide.md` §8.4)
- **ทุกหน้ามี Layout Canva + Prompt หน้านี้กำกับ** — ไม่ใช่แค่เนื้อหา แต่ต้องบอกด้วยว่าจัดวางอย่างไรและใช้ prompt อะไรสั่ง Canva (ดูตัวอย่างการกรอกในแต่ละ Slide ด้านล่าง)

## กติกา Design System ใน Canva (ตั้งค่าครั้งเดียวใช้ทุกหน้า)

> กรอกตารางนี้ก่อนเริ่มพิมพ์เนื้อหาจริง — ค่าที่ตั้งในตารางนี้คือค่าคงที่ทั้ง deck ห้ามเปลี่ยนรายหน้า (อิงเกณฑ์จาก [`slide_generate_guide.md`](../guide/slide_generate_guide.md) §4–§5)

| รายการ | ค่าที่ใช้ |
|--------|-----------|
| ขนาดสไลด์ | Presentation 16:9 (1920×1080) |
| ฟอนต์ | **Sarabun** ทั้ง deck |
| Title | 36–40 pt หนา ซ้ายบน ตำแหน่งเดียวกันทุกหน้า + เลขหัวข้อ "1 ·" ถึง "9 ·" ตรงกับ Outline |
| Subtitle ใจความหลัก | 20–22 pt สีเทา ใต้ Title ทุกหน้าเนื้อหา — ไม่เกิน 12–15 คำ |
| bullet เนื้อหา | 20–24 pt ไม่เกิน 5 ข้อ/หน้า ข้อละ 1 บรรทัด/8–10 คำ |
| ตาราง | 18–20 pt หัวตารางหนา |
| caption/label | 16–18 pt · **ขั้นต่ำสัมบูรณ์ 14 pt ทุกกรณี** |
| สีหลัก (Primary) | {{เช่น น้ำเงินเข้ม `#1A56DB`}} — ยึดสีมหาวิทยาลัย/คณะถ้ามี |
| สีตาม role/actor (ถ้ามีหลาย role) | {{เช่น `#0D9488` นักศึกษา · `#7C3AED` อาจารย์ · `#DB7F1A` สถานประกอบการ — ใช้กำกับหัวข้อ/แถบสีของ Slide 5 และ Slide 10 ให้ผู้ฟังแยก role ได้ทันทีจากสี}} |
| สีสื่อความหมาย (สถานะ) | เขียว = ทำแล้ว/active, เทา = 🔜 dev in next version, {{สีหลัก}} = จุดเน้น, ส้ม/เหลือง = คำเตือน — ใช้ใน Slide 5, 8, 9 (ดูตารางเต็มใน `slide_generate_guide.md` §5) |

**กฎเหล็ก:** เนื้อหาล้น → ตัดหน้า (1/2)(2/2) ชื่อ/layout เดิม **ห้ามลดฟอนต์** · ตัดที่รอยต่อความคิด ห้ามตัดกลาง role/flow · รายละเอียดที่ตัดออกใส่ช่อง Notes ของ Canva

## Slide 1.5 · Outline (หน้าแสดงหัวข้อ)

> แทรกระหว่างหน้าปกกับหัวข้อแรก — ไม่นับเป็น 1 ใน 9 หัวข้อหลัก แต่**ต้องมีเสมอ**

**วัตถุประสงค์ของสไลด์:** ให้ผู้ฟังเห็นแผนที่การนำเสนอทั้งหมดก่อนเริ่ม และประกาศ "เลขหัวข้อ 1–9" ที่จะใช้กำกับทุกสไลด์ถัดไป

**Layout Canva — เลือก 1 จาก 2 แบบ:**

| แบบ | เหมาะกับ | ทำอย่างไร |
|-----|----------|-----------|
| **(ก) numbered list ตรง 9 ข้อ** | ทำเร็ว ไม่ต้องคิดจัดกลุ่ม | 2 คอลัมน์ (5+4) ตัวเลขวงกลม{{สีหลัก}} |
| **(ข) การ์ดกลุ่มช่วงเลข (แนะนำ — อ่านง่ายกว่า)** | อยากให้หน้า Outline ดูโปร่ง ไม่แน่นเป็น list ยาว | จัดเป็น 4 การ์ด ช่วงเลขติดกัน + ชื่อหัวข้อสั้นในการ์ดเดียวกัน เช่น `01–02` ที่มาและเหตุผล · ซอฟต์แวร์เปรียบเทียบ / `03–04` วัตถุประสงค์ · ขอบเขตโครงงาน / `05–06` เครื่องมือที่ใช้พัฒนา · System Overview / `07–09` Functional Requirements · แผนพัฒนา · Demo Mockup |

**ข้อความ (ตายตัว — เลขชุดนี้ต้องตรงกับเลขหน้าชื่อทุกสไลด์ถัดไปคำต่อคำ):**

1. ที่มาและเหตุผล
2. ซอฟต์แวร์ในท้องตลาด vs ระบบของเรา
3. วัตถุประสงค์
4. ขอบเขตโครงงาน
5. เครื่องมือที่ใช้พัฒนา
6. System Overview
7. Functional Requirements
8. แผนการพัฒนา 3 เฟส
9. Demo Mockup

**Script พูด (~15 วิ):** "วันนี้ขอนำเสนอ 9 หัวข้อ เริ่มจากที่มาของปัญหา ไปจนถึงตัวอย่างหน้าจอระบบ" — ไม่ต้องไล่อ่านทุกข้อ

**Prompt หน้านี้:** `สไลด์ Outline การ์ด 4 ใบ แต่ละใบมีช่วงเลขหน้าตัวใหญ่ด้านบนและชื่อหัวข้อ 2-3 บรรทัดด้านล่าง โทน{{สีหลัก}}-ขาว ฟอนต์ Sarabun`

**แหล่งข้อมูลอ้างอิง:** หัวข้อคงที่ตาม template นี้ (ไม่ต้องคิดใหม่)

## ภาคผนวก A — Prompt สำหรับสั่งสร้างใน Canva (เส้นทางหลัก)

> ใช้เมื่อกรอกเนื้อหาทุกหน้าด้านบนครบแล้ว — copy prompt หลักด้านล่างไปวางใน Canva Magic Design™ เพื่อขึ้นโครง แล้วแทนที่ข้อความด้วย "ข้อความที่ต้องพิมพ์" ของแต่ละ Slide ด้านบน (วิธีละเอียด ดู `slide_generate_guide.md` §8.1)

### วิธีใช้
1. **Canva Magic Design™:** พิมพ์ prompt หลักด้านล่าง → เลือก Presentation 16:9 → แทนที่ข้อความด้วยเนื้อหารายหน้าในไฟล์นี้
2. **หรือ Canva Docs → Convert to Presentation:** วางส่วน "ข้อความที่ต้องพิมพ์" ทีละหัวข้อแล้ว Convert
3. ภาพ mockup: เปิดไฟล์ HTML ใน `mockup/<module>/` ด้วยเบราว์เซอร์ → capture เต็มจอ → Uploads เข้า Canva
4. หน้า System Overview (Slide 7) วาดใน Canva เอง (กล่องกลาง + กล่องรอบ + ลูกศร) ตามสเปกที่กรอกไว้ด้านบน

### Prompt หลัก (วางใน Magic Design)

```
สร้างงานนำเสนอวิชาการภาษาไทย {{จำนวนหน้า}} สไลด์ ขนาด 16:9 สำหรับสอบโครงงานปริญญานิพนธ์
หัวข้อ "{{ชื่อโครงงาน (ไทย)}} ({{ชื่อโครงงาน (อังกฤษ)}})"

สไตล์: minimal สะอาด เป็นทางการ พื้นหลังขาว โทนสีหลัก{{สีหลัก}}
สีรอง (ถ้ามีหลาย role): {{ไล่รายการสี role ตามตาราง Design System}}
ฟอนต์: Sarabun ทั้งเล่ม — หัวเรื่อง 36-40pt ตัวหนา, เนื้อหา 20-24pt, ตาราง 18-20pt, ไม่มีข้อความใดเล็กกว่า 14pt
ทุกสไลด์เนื้อหามี: เลขหัวข้อ+ชื่อหัวข้อมุมซ้ายบน, บรรทัดใจความหลักสีเทาใต้หัวเรื่อง, เลขหน้ามุมล่างขวา

โครงสร้าง {{จำนวนหน้า}} หน้า:
1) หน้าปก — ชื่อโครงงานไทย/อังกฤษ ชื่อนักศึกษา อาจารย์ที่ปรึกษา
2) Outline — รายการ 9 หัวข้อ (การ์ดกลุ่มช่วงเลข หรือ numbered list)
3) ที่มาและเหตุผล — bullet 5 ข้อพร้อมไอคอน (บริบท ปัญหา ผลกระทบ)
4) ตารางเปรียบเทียบซอฟต์แวร์ ไม่เกิน 4 เกณฑ์ x 4 คอลัมน์ คอลัมน์สุดท้ายไฮไลต์
5) วัตถุประสงค์ — numbered list
6) ขอบเขต — การ์ดตาม role + แถบหมายเหตุล่าง
7) เครื่องมือพัฒนา — โลโก้เทคโนโลยี grid 2x2
8) System Overview — diagram กล่องกลาง + กล่องรอบ + ลูกศร
9) Functional Requirements — ตาราง + บรรทัดยอดรวม
10) แผนพัฒนา 3 เฟส — 3 คอลัมน์หัวสีเขียว/เหลือง/แดง
11...) Demo Mockup แยกตาม role — screenshot 3 ภาพ + หมายเลข + ลูกศร + caption ต่อหน้า
สุดท้าย) ปิดท้าย — ขอบคุณที่รับฟัง
```

### Prompt เสริม (Canva AI / Magic Write)

- ปรับหน้าเดียว: `ปรับสไลด์นี้ให้ bullet ไม่เกิน 5 ข้อ ข้อละบรรทัดเดียว ตัวอักษรไม่ต่ำกว่า 20pt ถ้าเกินให้แบ่งเป็น 2 หน้าชื่อหัวข้อเดิมต่อท้าย (1/2)(2/2)`
- ทำ diagram: `วาดแผนภาพกล่องกลางใหญ่ชื่อ {{module}} ล้อมด้วยกล่องเล็กที่เชื่อมต่อ ลูกศรพร้อมข้อความกำกับขนาด 16pt`
- ทำหน้า demo: `จัดวาง screenshot 3 ภาพขนาดเท่ากันเรียงนอน วงกลมหมายเลข 1 2 3 มุมบนซ้ายแต่ละภาพ ลูกศรเชื่อม caption 16pt ใต้ภาพ`
